bug 产生的原理

这些问题在同一个 python 会话中，不会有问题，但是不同回话中，就会出现随机顺序的问题。

这是因为 set 背后的原理是 hash 结构，hash 结构需要一个 hash 函数计算 hash 值, python 中 string 的 hash 值和这个 string 对象的内存地址有关 (暂未找到具体证据佐证)。因此即使是同一个字符，在不同的 python 会话中，也会有不同的值。
